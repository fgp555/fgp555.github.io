# 📚 Manual de Atención - Call Center

Bienvenido al manual operativo de gestión telefónica.  
Este documento centraliza los principales **temas recurrentes** para una atención eficiente, estandarizada y orientada a resultados.

---

## 📌 Estructura de Atención

Todos los flujos siguen este esquema base:

1. Bienvenida
2. Validación de datos
3. Sondeo
4. Empatía
5. Gestión / Solución
6. Ofrecimiento (si aplica)
7. Cierre
8. Encuesta
9. Despedida

## 🧠 Buenas Prácticas

- Mantener siempre un tono cordial y profesional
- Escuchar activamente al cliente
- Reformular para confirmar entendimiento
- Aplicar empatía en todo momento
- Ofrecer soluciones claras y concretas
- Detectar oportunidades comerciales

---

## 📈 Indicadores Clave (KPIs)

- AHT (Tiempo promedio de atención)
- FCR (Resolución en primer contacto)
- NPS / Encuesta de satisfacción
- Tasa de retención
- Conversión comercial
